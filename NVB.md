# 制作指示書：NARAHA VERO BASE 公式LP

## 1. プロジェクト概要
* **イベント名**: NARAHA VERO BASE（ナラハ・ヴェロ・ベース）
* **開催日**: 2026年6月13日(土)・14日(日)
* **会場**: 福島県楢葉町 天神御崎公園
* **コンセプト**: 「自転車とキャンプで、地域に新しい『人の流れ』と『物語』をつくる」
* **特徴**: 
    * 日本唯一の「自転車×キャンプ」旅フェスティバル。
    * 「ツールド福島」との同時開催によるサイクリストの聖地化。
    * 競技ではなく「楽しむ・語らう」ことを目的としたCHILLな雰囲気。
* **ターゲット層**: 20代〜60代と幅広い（ファミリー、カップル、ソロ）。自転車×キャンプに興味がある初心者〜中級者。「楽しむ」ことが目的のためマナーが良い層。

## 2. 技術スタック（確定）
* **Frontend**: Astro (SSG) - 高速・軽量・モバイルファースト
* **Headless CMS**: Sanity - ニュース投稿システム
* **Styling**: Tailwind CSS - レスポンシブ設計
* **Deployment**: Cloudflare Pages - 安定・高速配信
* **Image**: Astro AssetsによるWebP最適化

## 3. 開発フェーズ・ロードマップ

### Phase 1：基盤・インフラ構築
* GitHubアカウント（`TokyoSports-UT9-2618` 配下）にリポジトリを作成し、プロジェクトを初期化。
* AstroとTailwind CSSの導入。
* Sanityのセットアップとスキーマ定義（news: title, date, content, image）。
* Cloudflare Pagesへの自動デプロイ環境の構築。
* **Goal**: URLにアクセスして「Hello World」とSanityのテストデータが表示されること。

### Phase 2：ニュース投稿システムの完成
* Sanity Studio（管理画面）のデプロイ。
* Astro側でSanity APIからのデータ取得と一覧表示の実装。
* モバイルからの投稿・閲覧を想定したUI調整。
* **Goal**: スマホから投稿したニュースがサイトに即座に反映されること。

### Phase 3：メインLP実装（コンテンツ制作）
* 提供資料に基づいた以下のセクション実装。
    * Hero: 絶景を活かしたメインビジュアルと開催概要。
    * Introduction: Not a Race, The Vibe, Inclusivity。
    * 3つの体験: EXHIBITION, ACTIVITY, CAMPING。
    * 楢葉・天神御崎公園の魅力紹介と、ツールド福島との連携シナジー。
* モバイルファーストなインタラクションの実装。
* **Goal**: デザイン・文言が全て揃ったLPの完成。

### Phase 4：公開・最適化
* 独自ドメインの設定。
* SEO（Meta/OGP）設定。
* PWA対応（ホーム画面追加機能）。
* **Goal**: 正式公開とパフォーマンススコア（Lighthouse）90点以上の達成。

## 4. 運営・フッター情報（Contact）

## 5. AIエージェントへの特別指示
* 「WordPress不使用」「モバイルファースト」「爆速」を常に優先すること。
* Sanityのスキーマは、運用者が直感的に入力できるよう日本語のラベルを設定すること。
* AstroのImageコンポーネントを使い、ヒーロー画像等の大きなアセットを徹底的に最適化すること。
* 米西海岸のような自由でCHILLな雰囲気を、カラーパレットとタイポグラフィで表現すること。
